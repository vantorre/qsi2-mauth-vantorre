User stories :

    As an anonymous user, I want to create an account so that I could login.    OK
    As an anonymous user, I want to login so that I could access authenticate actions.  OK
    As an authenticate user, I want to create a group so that I could be a group admin.
    As a group admin, I want to add or delete user so that I could make my group grow.
    As a group member, I want to get a list of all so that I could list most recent posts.
    As an anonymous user, I want to get a list of all groups so that I could ask to join one of them.
    As an API operator, I want secure my apis so that I could deliver a good service to end-users.



end points a implementer:
Public

POST   /api/v1/users/login "log a user" OK
POST   /api/v1/users/  "create a user"  OK

Authenticated

GET    /api/v1/users/  "get info of logged user"    OK
PUT    /api/v1/users/  "modify info of logged user" OK
DELETE /api/v1/users/  "delete the logged user"     OK
